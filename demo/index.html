Path Check (quick self-test)

  1. Right-click Open Notes in New Tab → Open Link in New Tab.
  2. If you get a 404, look at the URL bar. It should end with /outputs/notes.html (not /demo/outputs/...).
  3. From the repo root, confirm the file exists:
    ls outputs/notes.html outputs/deck.csv outputs/notes.pdf dist/lecture_easyread.zip
  4. If serving the root, links must start with ../ (as above). If you serve the demo folder directly, remove the ../ so they become outputs/... and dist/....

How to serve locally

Serve the repo root (recommended):

poetry run python -m http.server 8090

Open: http://localhost:8090/demo/

What’s included

  • outputs/notes.html – accessible notes viewer
  • outputs/notes.pdf – printable PDF
  • outputs/deck.csv – flashcards (CSV)
  • outputs/deck.apkg – Anki deck (if generated)
  • dist/lecture_easyread.zip – shareable bundle

Study the cards

  • Open Anki → File → Import → choose deck.apkg.
  • Or import deck.csv (map first field = Front, second = Back).
